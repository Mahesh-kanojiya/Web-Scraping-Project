# Web-Scraping-Project

<h1 align="center">🏏 IPL 2026 Auction Web Scraping</h1>

<p align="center">
A Python-based web scraping project that extracts IPL 2026 Auction player data from the official IPL website and converts it into a structured dataset for analysis.
</p>

<hr>

<h2>📌 Project Overview</h2>

<p>
This project uses Python web scraping techniques to collect IPL 2026 Auction information such as player details, nationality, role, team, and auction price from the IPL Auction website. The scraped data is cleaned and stored in a structured format, making it suitable for data analysis and visualization.
</p>

<h2>✨ Features</h2>

<ul>
    <li>Scrapes IPL 2026 Auction player data.</li>
    <li>Extracts structured player information.</li>
    <li>Cleans and processes raw HTML data.</li>
    <li>Stores data in CSV format.</li>
    <li>Easy to modify for future IPL seasons.</li>
</ul>

<h2>🛠️ Tech Stack</h2>

<ul>

</ul>
